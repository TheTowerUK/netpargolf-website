# NetParGolf Website

Official website for NetParGolf.

## About NetParGolf

NetParGolf is a golf scoring and handicap management app for iOS and Android.

Features include:

* Live scoring
* Course setup and management
* Competition handicap calculations
* Scorecards and round history
* Stableford and matchplay support
